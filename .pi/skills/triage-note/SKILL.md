# Triage Note Workflow

1. Parse the free‑text presentation and identify the **Chief Complaint**.
2. Summarize the relevant **History** (history of present illness, relevant past medical history, medications, allergies, social history) in a concise paragraph.
3. If **Vitals** are mentioned, list them in a table or bullet list; otherwise omit the section.
4. Highlight any **Red Flags** that warrant urgent evaluation.
5. Provide a **Suggested Acuity** (e.g., “Urgent – requires immediate assessment”) with brief reasoning, and note that it is a suggestion needing clinician confirmation.
6. List any **Missing Information** that should be obtained during the encounter.

Keep each section brief and use the exact headings: *Chief complaint*, *History*, *Vitals*, *Red flags*, *Suggested acuity*, *Missing information*.
